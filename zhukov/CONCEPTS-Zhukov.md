### 9. Интерфейс 
Интерфейс — структура программы или синтаксиса, определяющая отношение с объектами, объединёнными некоторым поведением. При проектировании классов разработка интерфейса тождественна разработке спецификации (множества методов, которые каждый класс, использующий интерфейс, должен реализовывать).

Пример создания интерфейса на Python.

Необходимо установить зависимости
```shell
cd .. && source env/bin/activate && pip3 install -r requirements.txt
```

```python
from abc import ABC, abstractmethod


class IComponent(ABC):
    """
    Базовый интерфейс Компонента определяет поведение, которое изменяется
    декораторами.
    """

    @abstractmethod
    def operation(self) -> str:
        """
        Основной метод, который будут модифицировать декораторы.
        Должен быть реализован в конкретных компонентах.
        """
        raise NotImplementedError("Метод должен быть реализован в подклассе")
        pass
```
