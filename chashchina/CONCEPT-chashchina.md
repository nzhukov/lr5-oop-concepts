# Создание объектов

## Определение 

Создание объектов — центральная концепция ООП, где программа моделируется как взаимодействие экземпляров (объектов), созданных на основе шаблонов (классов). Класс определяет структуру (данные/поля) и поведение (методы), а объект инициализируется через конструктор, получая уникальное состояние. Это упрощает код и позволяет имитировать реальные сущности

_ _ _

## Реализация 

``` python
class TemplateClass:
    """
    Шаблонный класс для демонстрации создания объектов
    """
    
    # Конструктор (вызывается при создании объекта)
    def __init__(self, parameter1, parameter2):
        """
        Инициализация атрибутов объекта
        - self: ссылка на создаваемый объект
        - parameter1, parameter2: входные параметры
        """
        self.attribute1 = parameter1
        self.attribute2 = parameter2
        self.attribute3 = None  # Атрибут со значением по умолчанию
    
    # Обычный метод
    def some_method(self):
        """Пример метода, работающего с атрибутами объекта"""
        return f"Значение атрибута1: {self.attribute1}"
    
    # Метод с изменением состояния
    def update_attribute(self, new_value):
        """Изменение атрибута объекта"""
        self.attribute3 = new_value


# --- СОЗДАНИЕ ОБЪЕКТОВ (ЭКЗЕМПЛЯРОВ) ---

# Способ 1: Стандартное создание
object1 = TemplateClass("значение1", "значение2")

# Способ 2: Создание с разными параметрами
object2 = TemplateClass(123, 456)

# Способ 3: Пустой объект (если конструктор это позволяет)
# object3 = TemplateClass()  # Требует доработки конструктора

# Доступ к атрибутам
print(object1.attribute1)
print(object2.attribute2)

# Вызов методов
result = object1.some_method()
object2.update_attribute("новое значение")

# Проверка типа объекта
if isinstance(object1, TemplateClass):
    print("object1 является экземпляром TemplateClass")
```

***Ключевые особенности Python при создании объектов***

| Особенность | Описание |
|-------------|----------|
| `self` | Явный первый параметр всех методов (ссылка на текущий объект) |
| `__init__` | Конструктор — вызывается автоматически после создания объекта |
| `__new__` | (реже) Метод, управляющий самим процессом выделения памяти под объект |
| Динамичность | Можно добавлять/удалять атрибуты у уже созданного объекта |
| Отсутствие `new` | Объекты создаются вызовом класса как функции: `ClassName()` |
| `isinstance()` | Функция для проверки принадлежности объекта классу |
