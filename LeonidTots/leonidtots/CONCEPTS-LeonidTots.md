# Композиция

## Определение

Композиция — это способ построения классов, при котором один объект содержит другой объект как часть своей структуры.

Композиция показывает отношение «имеет» (has-a relationship).

Например:
- Машина имеет двигатель
- Компьютер имеет процессор

---

## Особенности композиции

- Один класс использует другой класс
- Объекты могут работать совместно
- Улучшается переиспользование кода
- Композиция часто предпочтительнее наследования

---

## Особенности Python

| Особенность               | Описание                                   |
|---------------------------|--------------------------------------------|
| Динамическая типизация    | Можно передавать любые совместимые объекты |
| Простое создание объектов | Объекты легко вкладываются друг в друга    |
| Duck Typing               | Важна реализация методов, а не тип         |

---

## Пример на Python

```python
class Engine:
    def start(self):
        return "Двигатель запущен"


class Car:
    def __init__(self):
        self.engine = Engine()

    def drive(self):
        return self.engine.start()


car = Car()
print(car.drive())
